    • GitHub Pages - Hospedagem de Site Estático

    O GitHub Pages é uma maneira fácil de hospedar sites estáticos diretamente de um repositório no GitHub. Aqui está como fazer:

    Hospedando um Site Estático

    1. Crie um repositório no GitHub (por exemplo: seu-usuario.github.io).
    2. Adicione seus arquivos HTML, CSS, imagens e outros recursos à estrutura do repositório.
    3. Vá para as configurações do repositório, role até a seção "GitHub Pages".
    4. Escolha o branch que contém o código do site (geralmente, master).
    5. Clique em "Save" para ativar o GitHub Pages.
    6. Seu site estará acessível em https://seu-usuario.github.io.

    Adicionando um Site em React

    Para adicionar um site em React ao GitHub Pages:

    1. Crie um aplicativo React usando npx create-react-app nome-do-app.
    2. Navegue até a pasta do aplicativo e faça as alterações necessárias.
    3. No package.json, adicione "homepage": "https://seu-usuario.github.io/nome-do-app".
    4. Faça o commit e o push das alterações para o repositório do GitHub.
    5. Ative o GitHub Pages como mencionado acima.
    6. Seu site React estará acessível em https://seu-usuario.github.io/nome-do-app.
