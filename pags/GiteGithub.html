    O Git e o GitHub são ferramentas essenciais para o desenvolvimento de software colaborativo. Aqui está um tutorial básico:

    O que é Git?

    O Git é um sistema de controle de versão que permite rastrear alterações em arquivos ao longo do tempo.

    O que é GitHub?

    O GitHub é uma plataforma de hospedagem de código que usa o Git. Ele permite que equipes colaborem em projetos e compartilhem código.

    Configurando Chaves SSH

    Configurar chaves SSH ajuda a autenticar com segurança no GitHub:

    1. Gerar uma chave SSH: ssh-keygen -t rsa -b 4096 -C "seu-email@example.com"
    2. Adicionar a chave ao SSH-Agent: eval "$(ssh-agent -s)"
      ssh-add ~/.ssh/id_rsa
    3. Copiar a chave pública: clip < ~/.ssh/id_rsa.pub (ou use a chave em ~/.ssh/id_rsa.pub)
    4. No GitHub, vá para "Settings" > "SSH and GPG keys" > "New SSH Key" e cole a chave.

    Criando e Clonando um Repositório

    Para criar e clonar um repositório:

    1. Criar um novo repositório no GitHub.
    2. Clone o repositório para sua máquina: git clone https://github.com/seu-usuario/seu-repositorio.git
    3. Adicione arquivos e faça commits: git add .
      git commit -m "Primeiro commit"
    4. Envie as alterações de volta ao GitHub: git push origin master
